A proposta do projeto é fazer uma biblioteca digital, onde pequenos autores possam publicar suas obras para o publico, eles conseguem gerenciar os livros, editar e fazer outros uploads.

Foi criado também uma página pra gerenciar os usuários, mas o acesso seria limitado ao criador, precisa de alguns ajustes de acordo com a ideia inicial, mas já está rodando perfeitamente.

Pra rodar o projeto primeiramente é preciso criar o banco de dados e as tabelas que estão na pasta SQL, seguindo a sequência da mesma, na pagina inicial vc encontra os livros que já foram feitos uploads, tem a opção ler e salvar, não acontece nada a clicar em ler pq são meramente ilustrativos, e pra vc salvar/favoritar é preciso fazer login.

Na aba superior direita vc pode criar conta e logar, logando com o perfil "admin" e senha "admin", vc tem acesso ao gerenciador de usuário e de livros, pelo qual vc pode deletar criar e editar, tanto livro quanto usuário.

Esse é o básico do projeto.
